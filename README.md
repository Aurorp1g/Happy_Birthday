# 生日快乐网站模板项目

## 🎉 项目介绍

这是一个精美的生日快乐祝福网站模板，可以通过简单的配置自定义祝福内容、图片和音乐，为亲朋好友创建个性化的生日祝福网站。

## 🚀 功能特性

- **个性化定制**：通过配置文件轻松修改祝福文字、图片和音乐
- **多屏滚动展示**：采用全屏滚动设计，多屏展示不同祝福内容
- **音乐自动播放**：每个页面可配置专属背景音乐
- **图片自适应**：图片自动缩放和居中显示，确保良好视觉效果
- **登录验证功能**：可设置简单密码保护，增加惊喜感
- **响应式设计**：适配不同设备屏幕尺寸
- **自动构建部署**：支持一键构建和部署到GitHub

## 🛠 技术栈

- Node.js
- Plop.js（代码生成工具）
- Handlebars（模板引擎）
- YAML（配置文件）
- jQuery
- CSS3 动画

## 📁 项目结构

```
Happy_Birthday/
├── config.yaml        # 配置文件（自定义内容）
├── dist/              # build 生成的静态网站文件
├── templates/         # 模板文件
│    ├── html/          # HTML模板
│    ├── js/           # JavaScript模板
│    ├── css/           # CSS模板
│    ├── img/           # 图片模板
│    ├── music/         # 音乐模板
│    ├── woff/          # 字体文件
│    └── index.hbs      # 入口HTML模板
├── package.json       # 项目依赖配置文件
└── plopfile.js.txt     # 构建配置文件(txt)
```

## 📋 快速开始

### 安装依赖

```bash
npm install
```

### 自定义配置

编辑 `config.yaml` 文件，根据需要修改以下内容：

- `title`: 网站标题
- `To_name`: 祝福对象的名字
- `from_name`: 送祝福的名字
- `crypto`: 登录密码（默认：1129）
- `font_animation`: 文字动画配置
- `music`: 页面背景音乐配置
- `screens`: 各屏幕的文字和图片内容

### 生成网站

```bash
npx plop g
```

### 本地预览

生成的网站文件在 `dist` 目录下，可使用任何静态文件服务器预览：

```bash
cd dist
npx http-server
```

或者直接在 `dist` 目录下双击 `index.html` 打开预览。


## 🎨 自定义指南

### 修改祝福文字

在 `config.yaml` 文件中的 `screens` 部分，修改对应的 `message` 数组内容。

### 更换图片

1. 将新图片放在 `templates/img/` 目录下
2. 在 `config.yaml` 中更新图片路径

### 更换音乐

1. 将新音乐文件放在 `templates/music/` 目录下
2. 在 `config.yaml` 中更新音乐路径

## 💡 提示

- 图片建议使用适合网页显示的尺寸和格式（如JPG、PNG）
- 音乐文件建议使用MP3格式
- 可通过修改模板文件进一步自定义页面样式
